# Dataset-cleanup-and-analysis #

A Python project to clean messy data, store it in SQL, and utilise Power BI to evaluate before and after metrics.



### Features ###

- Python/Pandas to load and clean datasets by correcting missing values, duplicates, inconsistent categories, date formats, and outliers.

- SQL to store the finalised dataset and calculate metrics of before and after versions of the dataset.

- BI visuals to graphically display before and after differences, as well as indentifying KPIs such as average order value, conversion rate, or total revenue.


### Tech stack ###

- Python/Pandas
- SQL
- Power BI
